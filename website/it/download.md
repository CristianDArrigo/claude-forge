## Requisiti di Sistema

| Requisito | Minimo |
|-----------|--------|
| **Sistema Operativo** | Windows 10 (64-bit) o successivo |
| **RAM** | 4 GB |
| **Spazio su Disco** | 200 MB disponibili |
| **Dipendenze** | Claude Code CLI installato |

## Prerequisiti

Prima di usare Claude Forge, devi avere **Claude Code CLI** installato sul tuo sistema.

### Installare Claude Code CLI

1. Visita la [documentazione di Claude Code CLI](https://docs.anthropic.com/claude-code)
2. Segui le istruzioni di installazione per la tua piattaforma
3. Verifica l'installazione eseguendo `claude --version` nel terminale

## Procedura di Installazione

1. Scarica l'installer dal link sopra
2. Esegui `Claude Forge Setup x.x.x.exe`
3. Segui la procedura guidata
4. Avvia Claude Forge dal menu Start o dal collegamento sul desktop

## Aggiornamenti

Claude Forge controlla automaticamente gli aggiornamenti all'avvio. Quando una nuova versione e disponibile, ti verra chiesto di scaricarla e installarla.

Puoi anche controllare manualmente gli aggiornamenti visitando la pagina [GitHub Releases](https://github.com/CristianDArrigo/claude-forge/releases).

## Risoluzione Problemi

### Claude CLI non trovato

Se Claude Forge non trova il Claude Code CLI:

1. Apri le Impostazioni (icona ingranaggio in alto a destra)
2. Vai alla sezione "Generale"
3. Imposta il percorso corretto dell'installazione di Claude CLI

### Problemi di permessi

Se incontri errori di permessi:

1. Esegui l'installer come Amministratore
2. Scegli una cartella di installazione diversa se necessario
3. Assicurati che l'antivirus non blocchi l'applicazione
