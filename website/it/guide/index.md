# Introduzione

**Claude Forge** e un'applicazione desktop che fornisce un'interfaccia grafica per Claude Code CLI. E progettato per sviluppatori che vogliono sfruttare la potenza dello sviluppo assistito da AI con un'interfaccia elegante ed efficiente.

## Cos'e Claude Forge?

Claude Forge racchiude il Claude Code CLI in una bella app desktop basata su Electron, aggiungendo funzionalita come:

- **Gestione multi-progetto** - Lavora su piu progetti contemporaneamente
- **Esecuzione visuale dei task** - Vedi le risposte dell'AI in tempo reale con output formattato
- **Tracciamento commit** - Traccia tutte le modifiche ai file fatte dall'AI
- **Integrazione Git** - Gestisci le operazioni git senza uscire dall'app
- **Agenti personalizzati** - Crea assistenti AI specializzati per diversi task

## Perche usare Claude Forge?

### Per Sviluppatori Individuali

- **Workflow piu veloce** - Niente piu passaggi tra terminale e IDE
- **Migliore visibilita** - Vedi esattamente quali modifiche sta facendo l'AI
- **Organizzazione** - Mantieni progetti e conversazioni organizzati
- **Cronologia** - Naviga i prompt precedenti con le frecce

### Per i Team

- **Configurazioni condivise** - Esporta e condividi le impostazioni degli agenti
- **Esperienza consistente** - Stessa interfaccia per tutti i membri del team
- **Traccia di controllo** - Traccia tutte le modifiche generate dall'AI

## Architettura

Claude Forge e costruito con tecnologie moderne:

| Componente | Tecnologia |
|------------|------------|
| Framework | Electron 28 |
| Frontend | React 18 + TypeScript |
| Stili | CSS Variables |
| Stato | Zustand |
| Build | Vite + electron-builder |
