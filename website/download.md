## System Requirements

| Requirement | Minimum |
|-------------|---------|
| **OS** | Windows 10 (64-bit) or later |
| **RAM** | 4 GB |
| **Storage** | 200 MB available space |
| **Dependencies** | Claude Code CLI installed |

## Prerequisites

Before using Claude Forge, you need to have **Claude Code CLI** installed on your system.

### Installing Claude Code CLI

1. Visit the [Claude Code CLI documentation](https://docs.anthropic.com/claude-code)
2. Follow the installation instructions for your platform
3. Verify the installation by running `claude --version` in your terminal

## Installation Steps

1. Download the installer from the link above
2. Run `Claude Forge Setup x.x.x.exe`
3. Follow the installation wizard
4. Launch Claude Forge from the Start menu or desktop shortcut

## Updating

Claude Forge will automatically check for updates on startup. When a new version is available, you'll be prompted to download and install it.

You can also manually check for updates by visiting the [GitHub Releases](https://github.com/CristianDArrigo/claude-forge/releases) page.

## Troubleshooting

### Claude CLI not found

If Claude Forge can't find the Claude Code CLI:

1. Open Settings (gear icon in the top-right)
2. Navigate to the "General" section
3. Set the correct path to your Claude CLI installation

### Permission issues

If you encounter permission errors:

1. Run the installer as Administrator
2. Choose a different installation directory if needed
3. Make sure antivirus software isn't blocking the application
